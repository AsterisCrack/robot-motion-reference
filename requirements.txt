#
# Isaac Lab bundles: numpy, torch, pxr (Pixar USD), and most other heavy deps.

#   https://isaac-sim.github.io/IsaacLab/main/source/setup/installation/index.html
#
# After installing, activate Isaac Lab's Python environment:
#   source /path/to/isaaclab/.venv/bin/activate

# --- 3. tkinter (system package — NOT pip-installable) ---
#
# preview_animation.py uses tkinter for the interactive tuning panel.
# tkinter is part of the Python standard library but requires a system package
# on most Linux distributions:
#
#   Ubuntu / Debian:  sudo apt install python3-tk
#   Arch Linux:       sudo pacman -S tk
#   macOS:            included with the official python.org installer
#   Windows:          included with the official python.org installer
#
# If Isaac Lab ships its own CPython build without tkinter, you may need to
# rebuild Python with --enable-framework (macOS) or install python3-tk and
# point Isaac Lab's venv at the system tk libraries.
